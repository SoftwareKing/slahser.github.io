## 基本命令: 

```
sudo /etc/init.d/oneapm-ci-agent start/stop/restart
sudo yum erase oneapm-ci-agent
```  

## 配置文件路径: 

```
/etc/oneapm-ci-agent/conf.d 
``` 

## 检查oneapm探针内置服务状态: 

```
//他们也是使用supervisor进行脚本管理的哦~
supervisorctl -c /etc/oneapm-ci-agent/supervisor.conf 
``` 

## 配置文件: 

zookeeper.yaml 

```
init_config:
instances:
  - host: localhost
    port: 2181
    timeout: 3
``` 
